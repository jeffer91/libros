# Generador de Libros y Guías

App local en Electron + React para generar libros de asignatura y guías de formación práctica desde Excel, con apoyo de Gemini y salida final en Word.

Estado actual del desarrollo por bloques:

- Bloque 0: base del proyecto.
- Bloque 1: menú y pantallas base.
- Bloque 2: base local y almacenamiento.
- Bloque 3: carga real de Excel.
- Bloque 4: lectura inteligente de Excel.
- Bloque 5: motor académico inteligente.
- Bloque 5.1: corrección de flujo y conexión general.
- Bloque 6: conexión real con Gemini.
- Bloque 7: generación base del libro.

Pendiente: generación de guía, referencias reales, APA 7 avanzado, motor Word, verificación final, descarga e IA local.
